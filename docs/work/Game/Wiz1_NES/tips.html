Wizardry(NES) 解析 - 小ネタ

解析によって判明した様々な興味深い事実についてまとめておく。

とりあえず箇条書き:

  • 職業、種族によって罠にかかった際の罠回避率、ブレスダメージ軽減率などが 異なる(運勢値の項目を参照。ただし、高レベルにな ると関係なくなる)
  • PTキャラのACは打撃攻撃回避率に影響しない (敵からの打撃攻撃の項目を参照。なお、 敵キャラの場合はACが低いほど攻撃が当たりにくい)
  • PTキャラが行うPARRYには意味がある(打撃攻撃回避率が若干上がる)
  • MALIKTO, TILTOWAITなどの呪文も属性を持っており、モンスターの種類によっ てはダメージを軽減される
  • MORLIS/MAMORLISによるAC上昇量は4ではなく3である
