Wizardry(NES) 解析 - 逃走

戦闘中のRUNコマンドによる逃走について。

逃走成功率

逃走成功率はPTの人数、PTの合計LV, 敵の合計LVに依存し、次式で表される(百分 率):

(逃走成功率) = 75 + (PT人数ボーナス) + (LVボーナス) (%)
※100以上ならば必ず逃走できる

PT人数ボーナスは、PT人数(死亡などのステータス異常の有無を問わない)が

  • 1人の場合 +15
  • 2人の場合 +10
  • 3人の場合 +5
  • 4人以上の場合 0

また、(PTの合計LV) >= (敵の合計LV) の場合、LVボーナス +20 が与えられる (敵についてはグループ単位でなく個体単位での合計)。ただし、PT, 敵ともステータ ス異常にかかっている者のLVは加えない。

なお、以下のイベント戦闘では絶対に逃走できない:

  • B3F (13,17) で発生する玄室エンカウント(このマスはイベントマスで、少し強 い敵が出現する)
  • コントロールセンター(B4F (11-12,12-14))でのLVL 7 FIGHTERパーティー戦
  • WERDNA戦(B10F (17,3))

逃走成功時の処理

現座標と直前座標が入れ替わる。また、エナジードレインを受けた者がいる場合、 そのための処理も行われる(詳細未解析)。
