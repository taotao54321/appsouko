Wizardry(NES) 解析

NES(USA)版 Wizardry #1の解析資料。まだまだ手探り状態なので誤りが含まれて いる可能性あり。FC(日本)版にも適用できる部分が多いが、全くの同一仕様ではない ので注意。

NES版では僧侶はCLERIC, 司教はWIZARDと表記されるが、ここではPRIEST, BISHOP と表記する。その他の用語も一部他機種のものに置き換えているので注意(例: MAGE's MISERY → MAGE BLASTER など)。

特に断らない場合、除算は小数点以下切り捨てとする。たとえば 5/2 は2。ただ し確率に関する記述は別(51/256 などと書かれていたら約20%ということ。分子と分 母それぞれの内部での除算はやはり小数点以下切り捨てとする)。

特に断らない場合、確率に関しては乱数値総当たり調査までは行っていないので、 確率に関する記述はあくまで目安として考えてほしい。

キャラステータスのうちPOISON, 沈黙は別に管理されているので、特に断らずに 「ステータス異常」と言った場合はPOISONおよび沈黙を含まないものとする。

解析によって判明した事実のうち、特に興味深いものは 小ネタとして別にまとめておく。
