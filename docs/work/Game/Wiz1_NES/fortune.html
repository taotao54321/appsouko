Wizardry(NES) 解析 - 運勢値

各PTキャラは「運勢値」という隠しパラメータを持っており、この値が罠にかかっ た際の罠回避率、ブレスダメージ軽減率などに影響する。運勢値は5種類存在する。 各運勢値を「運勢値1」「運勢値2」のように表記する。

基本的に運勢値が小さい方が各種確率判定において有利になるが、例外もある (MAGE BLASTER, PRIEST BLASTERは運勢値が小さいほど被害を受けやすい)

運勢値の算出

運勢値nは、キャラのLV, LUC, 職業、種族に依存する。まず、次式で値 A(n) を 求める:

A(n) = 19 - LV/5 - LUC/6 - (運勢値nの職業補正値) - (運勢値nの種族補正値)
※LVは 0xFF を超えていたら 0xFF に補正
※A(n) < 0 の場合は A = 0 とする

ここで、職業補正値は以下の通り:

職業運勢値1運勢値2運勢値3運勢値4運勢値5
FIG 3 0 0 0 0
MAG 0 0 0 0 3
PRI 0 3 0 0 0
THI 0 0 0 3 0
BIS 0 2 2 0 2
SAM 2 0 0 0 2
LOR 2 2 0 0 0
NIN 3 2 4 3 2

種族補正値は以下の通り:

種族運勢値1運勢値2運勢値3運勢値4運勢値5
HUM 1 0 0 0 0
ELF 0 0 2 0 0
DWA 0 0 0 4 0
GNO 0 2 0 0 0
HOB 0 0 0 0 3

以上で求めた A(n) に1を加えたものが運勢値nとなる。
