グループ番号、キャラ番号

敵グループ番号は上から順に [1, 4]。便宜的にPTを第0グループとして扱うこと もある。

戦闘に参加するPT/敵キャラには番号が振られている。キャラ番号は、PT1～6人目、 敵第1グループ1～9体目、敵第2グループ1～9体目、…の順に [0, 41]。

キャラ番号対応するキャラ
[0, 5]PT1～6人目
[6, 14]敵第1グループ1～9体目
[15, 23]敵第2グループ1～9体目
[24, 32]敵第3グループ1～9体目
[33, 41]敵第4グループ1～9体目

グループが全滅したりグループ内の敵が死亡したりした場合も、敵を「前に詰め る」ような処理は行われない。つまり、敵グループの番号は常に不変であり、一度出 現した敵のキャラ番号も常に不変である。

戦闘中の行動

PTキャラが行うPARRYは打撃攻撃回避率を上げる効果がある (敵からの打撃攻撃を参照)。敵が行う PARRYは何の効果も持たない(何もしないことと同義)。
