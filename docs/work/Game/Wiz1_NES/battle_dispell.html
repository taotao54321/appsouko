Wizardry(NES) 解析 - ディスペル

DISPELLコマンドの仕様および成功率について。

仕様

不死系以外のモンスターはDISPELLできない。また、ステータス異常にかかってい る敵もDISPELLできない。

DISPELL成功判定は対象グループ内の敵1体ごとに行われる。

DISPELL成功率

不死系モンスターに対してDISPELLを行った場合、ステータス異常にかかっていな い敵1体に対するDISPELL成功率は、術者と対象モンスターのLVに依存する。ただし最 低成功率5%が保証されている。

まず、「ディスペル力」、「ディスペル抵抗力」を以下のように定義する:

(ディスペル力) = 5*(術者のLV) + 50
(ディスペル抵抗力) = 10*(対象モンスターのLV)

ディスペル力が 0xFF を超えた場合、ディスペル抵抗力によらずDISPELLは必ず成 功する。ディスペル力が 0xFF 以下の場合のDISPELL成功率は次式で表される(百分率):

(DISPELL成功率) = (ディスペル力) - (ディスペル抵抗力) (%)
※5未満の場合は5に補正
※100以上ならば必ずDISPELLできる

以上より、LV 42 以上のキャラならば、仕様の項の条件を満たしていればDISPELL は必ず成功する。

高いディスペル抵抗力を持つ不死系モンスターとしては、MAELIFIC(LV 25), VAMPIRE LORD(LV 20)などが挙げられる。
