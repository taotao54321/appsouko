Wizardry(NES) 解析 - 宝箱罠

各種宝箱罠の詳細。

「箱を開けた者」とは、宝箱コマンドメニューで INSPECT/DISARM/OPEN を実行し て罠にかかった者、という意味。

POISON NEEDLE

箱を開けた者のPOISON値を1増やす。

POISON値のオーバーフローチェックは行っていない。よって、POISON NEEDLEにか かり続けることでPOISON値をオーバーフローさせて0にすると毒が治る(現実的ではな いが)。

GAS BOMB

PT内の生存者がランダムでPOISON値1になる。1人当たりの確率はPTの先頭キャラ の運勢値4に依存し、次式で表される:

(毒を受ける確率) = (20-(PT先頭キャラの運勢値4)) / 20

CROSSBOW BOLT

箱を開けた者に (現在の階層)d8 のダメージ

EXPLODING BOX

PT内の生存者がランダムでダメージを受ける。

個々の生存者について 1/4 の確率でダメージを回避できる。回避できなかった場 合、以下のダメージを受ける:

  • 2/3 の確率で (現在の階層)d5
  • 1/3 の確率で (現在の階層)d8

STUNNER

箱を開けた者がPARALY状態になる。ただし元々PARALY以上のステータス異常にか かっていた場合は変化なし。

# PARALY以上のステータス異常の者はそもそも箱を開けられないはずだが…

TELEPORTER

同じ階のランダムな座標へ移動する([0, 19] の乱数を2回発生して座標を決める)。

なお、直前座標は更新されない。よって、TELEPORTERにかかった直後に一歩も動 かず方向転換し続けてエンカウントし、その戦闘から逃走すれば元の座標(玄室の手 前)に戻れる。

MAGE BLASTER

PT全員の職業、運勢値5をチェックし、

  • MAGEは (運勢値5)/20 の確率で麻痺し、そうでない場合は石化する
  • SAMURAIは (運勢値5)/20 の確率で何も起こらず、そうでない場合は麻痺する

いずれの場合も既により悪いステータス異常にかかっていたらそのまま。

なお、MAGE呪文を修得しているかどうかは関係ない。

PRIEST BLASTER

PT全員の職業、運勢値5をチェックし、

  • PRIESTは (運勢値5)/20 の確率で麻痺し、そうでない場合は石化する
  • BISHOPは (運勢値5)/20 の確率で何も起こらず、そうでない場合は麻痺する

いずれの場合も既により悪いステータス異常にかかっていたらそのまま。

なお、LORDは影響を受けない。また、PRIEST呪文を修得しているかどうかは関係 ない。

ALARM

ALARMフラグが立つ。ALARMフラグ自体については詳細未解析。
