Wizardry(NES) 解析 - 属性

属性と属性抵抗について。

属性

属性は以下のものがある(ビットフラグなので複合可能):

値名称
0x01友好
0x02火
0x04冷気
0x08毒
0x10吸収
0x20石化
0x40呪文
0x80(未解析)

キャラが属性抵抗を持つ場合、その属性を持つ攻撃に対するダメージ軽減などの 効果が得られる。PTキャラの属性抵抗は全所持アイテム(装備する必要はない)の属性 抵抗の論理和となる。敵キャラは種類ごとに固有の属性抵抗を持つ。

エンカウント時に友好属性抵抗を持つ敵がメイン敵となった場合、決して友好的 にならない。

無属性の攻撃呪文は存在しない。TILTOWAIT, MALIKTOなどの呪文は無属性ではな く友好属性であり、友好属性抵抗を持つ者にはダメージを軽減されてしまう。

PTキャラが打撃攻撃を受けた際の毒、麻痺、石化、ドレイン、クリティカルにつ いては全く関係ない種類の属性抵抗が参照される。詳細は 敵からの打撃攻撃を参照。
