TAS 関連

TAS (Tool-Assisted Speedrun) について色々。

ニコニコ動画にいくつかの動画をアッ プしています(ニックネームは TaoTao)。

機種名について: 原則として、日本版ならFC/SFC, 海外版ならNES/SNESを用いま す。

2009/08/17: これまでにアップしたFCMファイルについて、 fceutools 内の fcmoptim を用いてサイ ズ最適化を施しました(内容は変化なし)。もしFCMファイルを何らかの形で利用した い場合は新しい方を使うことをオススメします(初期ステートが無効化されているの で、変な事故などが起こりにくいはず)。
