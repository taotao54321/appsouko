シーザースパレス(GB) TAS

「GBのゲーム制覇しましょ」の 記念として作成。

22.62 (vbmファイル)

スロットの大ジャックポット(JP)を2回出すだけです。100万JP→500万JPルートも 考えられますが、それだと少し遅くなるようです。各リール値はコイン投入及びレバー 操作のタイミングで決定され、黒7になるのはフレームカウンタの256通りの値のうち 4通りのみです。よって、黒7が4つ揃う確率は 1/64^4 = 1/16777216 となります(た だしガチプレイでこれが達成された例があります)。

解析関係のことはGB制覇の過去ログを見れば色々書いてあります。JPスロットの リール値決定処理は ROM0:24F3, ブラックジャック(BJ)のカードシャッフル処理は ROM1:4042 にあります。

なお、このゲームはBJに必勝法が存在するため、ツールアシストなしでもクリア 可能です。BJのカードシャッフル結果は前回のデッキとフレームカウンタ(256通り) にのみ依存するので、前回のデッキを記録しておき、256通りのシャッフル結果をリ ストアップし、次回のデッキで最初に出てきたカードと照合すれば、それ以降のカー ドが予想できるようになります(GB制覇のWikiにツールがあります)。
