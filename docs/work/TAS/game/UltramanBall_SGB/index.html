ウルトラマンボール (SGB) TAS

わりと微妙なゲームですが、アイデアは面白いと思うのでTASさんにやってもらい ました。このゲームはFC『アルマジロ』の続編的な位置付けらしいですが、『アルマ ジロ』に比べて処理落ちは減ったものの面構成などはかなり適当になってしまった感 があります。

vbmファイル

ほとんど丸まりジャンプで移動することになるため、意外と追記回数が多くなり ました。

メモ

丸まり状態で空中にいるときの最高x速度は32で、基本的にはこの状態で移動する のが最速です。ただし、人型で歩くときはx速度は16で一定なので、静止状態からの 始動はこちらを使う方が速いです。

丸まりジャンプの上昇中に丸まり解除するとy速度が-48になるため、少しジャン プ高度が伸びます。

空中で丸まり状態に移行するとy速度が0にリセットされるため、空中でB連打する ことでほぼ浮遊状態にできます。また、丸まり状態で天井付近にいるときにBを押す と、そのフレームでの加速度を無効化できるため、B連打で多少滞空時間を稼げます。 なお、このとき天井が壊れるブロックの場合は壊れます。

丸まり状態で上昇中に丸まり解除すると、x速度の向きはそのままで絶対値が16に なります。これは衝突による跳ね返りよりも優先されるため、壊れるブロックに衝突 する瞬間に丸まり解除すれば速度反転を防げます。

ツタに掴まってから横移動すると一瞬だけx速度が48になるため、ツタの前で上と 右を交互に連打するとブーストが可能です。
