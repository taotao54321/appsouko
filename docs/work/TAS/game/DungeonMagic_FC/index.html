2008/10/11: FatRatKnight 氏によ る TAS (USA版、1:02.83) が TASVideos に提出されました(10/14 追記: publish されました)。10/08 版からの変更点は:

  • 食料を買うのでなく、薬を拾うことで必要な HP を賄う
  • トレス神殿 B2F に降りた直後で扉すり抜けを行う

拾う薬の選択も限界まで最適化されているので、これはもう完璧と言っていいレ ベルに達していると思います。何か新しいバグ技でも発見されれば別ですが…。ちな みに、FatRatKnight 氏は GameFAQs の FAQ 著者でもあります。

2008/10/08: FatRatKnight 氏に よる TAS (USA版、1:14.78) が TASVideos に提出されました。乱数周りがきち んと解析されているため、敵の動きを思うままに制御でき、ロスがほとんどなくなっ ています。ただし、

  • トレス神殿 B2F に降りた直後で扉すり抜けを行っていない
  • 薬を拾わず食料を買っている(買い物は比較的時間のかかる処理)

という点が気になったため、フォーラムでその旨を氏に伝えたところ、以上の点 を再検討して更新版を作成されるとのことです(10/08 版は cancel されました)。 更新版に期待。

2008/10/12 追記: 日本版はUSA版に比べてオープニングが短く、コマンド入力も 素早く行えるのですが、ポーズ中も敵が動く仕様なので、USA版のような敵の動きの 制御は難しそうです。この点が解決されない限りUSA版ほど速くはならないと思われ ます。

1:15.33 (fcmファイル → fcmoptimで最適化したもの)

攻略ルート:

  • ゼムサに入信
  • 聖剣トレスの神殿でトレス入手
  • ラストダンジョンのED部屋に入る

これ以外のルートはちょっと考えにくいと思います。なお、魔王を倒さなくても ED部屋に入ればクリアとなります。

使用したテクニック(バグ技?):

  • 攻撃魔法を発射し、着弾前にスペルを変更して非攻撃魔法(マップ表示魔法は 不可)を発動すると、ときどき魔法攻撃力が 255 になる(アンダーフロー?)。ただし、 LV0 かつトレスを所持していない状態でなければならない。未検証だが、最初に発 射する攻撃魔法は攻撃薬でも代用できると思われる
  • 敵の攻撃の数フレーム前にポーズをかけて即解除すると、敵の攻撃がキャンセ ルされる

以上のテクニックを使えば LV0 でもほとんどの敵を倒せます(ラストダンジョン の敵はトレス入手後なので無理)。トレス神殿は回避不能な敵もいるのでこれらのテ クニックは必須。

全然まじめに解析してませんが、とりあえず気付いたことだけメモ:

  • 敵と向かい合うとカーソルが勝手に攻撃に移ってしまうが、予めカーソルを魔 法に合わせておけば動かない
  • 扉が開いて通過できるようになるフレームは決まっている。おそらく 8F 間隔
  • ポーズ中も扉は開き続ける
  • 開いている/開きつつある扉の前にいるときは背面からの攻撃を受けない
  • 階段のすぐ目の前に扉がある場合、階段昇降時の画面暗転中に上キーを入力す れば扉を通り抜けられる(猶予は 1 フレームのみ)
  • マップを切り替えるタイミングによって敵の初期配置/動きが変化する
  • トレス神殿 B3F 北東は、初期配置によってはかなりスムーズに攻略できる。 ただし、B3F に到達したタイミングによっては理想の配置を実現するまでにかなり のフレーム待ちが必要になるかもしれない
  • 攻撃魔法を発射し、着弾前にスペルをより上位の攻撃魔法に入れ替える(発動 はしないでおく)と、着弾時のダメージが増すようだ
  • なぜかポーズができなくなることがある(ゼムサの魔法で天空にワープした直 後など)
