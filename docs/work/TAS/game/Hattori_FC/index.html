Takanawa氏の動画:

メモ

チクワと鉄アレイは合わせて最大255本出現するようです(ボーナスステージ残り 時間 $03F2 がデクリメントされると同時にどちらかが出現している模様)。また、 $03F2 は4Fごとにデクリメントされるようです。

影走りの術を使うと移動が2単位になるため、x座標は奇数または偶数に固定され ます。画面左端にぶつかることで奇数から偶数に変更できますが、逆はできないよう です。
