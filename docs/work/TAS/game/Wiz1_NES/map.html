Wizardry(NES) Map

とりあえずマップデータを機械的に処理した画像を置いておきます。

マップの形式はWe Love WIZARDRY for WonderSwanさんのものをパクってます。また、座標部分の画像を勝手に使わせて いただきました^^;

斜線部は玄室として扱われるマスで、この領域に扉を蹴破って進入するとブラン ドエンカウントが発生する可能性があります。なお、床データに関しては玄室とダー クゾーン以外は各階で数値の意味が違うので、数値(16進)をマス目に直書きしておき ます。
