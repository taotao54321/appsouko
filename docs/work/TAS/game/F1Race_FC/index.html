F1レース(FC) TAS

LV3最終面をターボなしで2周できるかどうかを検証する目的で作成 (「FCのゲーム制覇しましょ」 では6面でターボを発動して最終面2周しましたが、ターボなし2周は私の知る限り確 認されていません)。これによりターボなしでも2周は可能という結論が得られました が、今度は理論上3周可能かどうかという謎が残ってしまいました^^;

fm2ファイル(追記55120回。FCEUXは最終フレームの 入力が複製されるバグがあるようなので、末尾に無入力を追加済)

上の動画では残り時間があと22Fあれば3周できますが、2周目の乱数が悪く、敵車 をどのように調整してもロスが避けられないように見えるので、細部をこれ以上最適 化しても届かないと思われます。というか、仮に敵車が全く出なかったとしても届く かどうか微妙な感じです(気力が尽き果ててるので検証はしてませんが)。

基本的にメモリとにらめっこしながら最適化する以外に方法がないので、制作は 結構面倒です。自動操縦botのようなものが作れれば良いのですが、考慮すべき要素 がかなり多いため私の頭では無理そうです。
