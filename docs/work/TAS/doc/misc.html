TAS 関連の雑多なメモ

RPG のメニュー操作は同時押しの活用で数F短縮できることがある。

ボタン連打は無入力フレームを挟む必要があるが、同じ機能のボタンが2つ以上あ るならそれらを交互に押すことで無入力フレームを消去できることがある。

FCEU のメモリビューアから "Memory Poke" で直接メモリを書き換え られる。

FCEU の Lua スクリプトを実行するとなぜか 1 フレーム余計に進む。もしこれが 問題になるならスクリプトの最後でどこかのスロットにステートセーブしておくこと。

.fcm ファイルには ROM ファイルの MD5 (厳密には、.nes ファイルから 16 Byte の iNES ヘッダを取り除いた ROM バイナリの MD5)も記録される。なお、この MD5 は FCEU の Movie Replay ダイアログで表示されるが、その際各バイトの上位ニブル と下位ニブルが入れ替わって表示されてしまうので注意(0.98.28 で確認。md5.c 内 の md5_asciistr() のバグと思われる)。
