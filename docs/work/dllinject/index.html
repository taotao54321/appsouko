dllinject

DLL injectionを行うコマンドラインツールです。

ダウンロード: dllinject-0.1.0.tar.bz2

Windows XP SP3 (32bit) で動作確認しています。

一部のアンチウイルスソフトで警告が出るかもしれません (VirusTotalでの診断結果) が、実害はないと思います。心配な方はソースを読んで自ビルドしてください。

使い方

以下のようにターゲットプロセスとDLLを指定します:

  $ dllinject hoge.exe fuga.dll # EXEファイル名指定
  $ dllinject 123 fuga.dll      # プロセスID指定

EXEファイル名指定時に同名プロセスが複数あった場合、最初に見つかったものが 対象となります。
